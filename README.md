# pyrefly-loong64

This project builds [Pyrefly](https://github.com/facebook/pyrefly) for LoongArch with bundled binaries.

To get the extension, search for it in the extensions tab of VSCodium and install it directly.

To install the `.vsix` file offline, please get it from [releases](https://github.com/loongcodium/pyrefly-loong64/releases), and then install it with `codium --install-extension`.
